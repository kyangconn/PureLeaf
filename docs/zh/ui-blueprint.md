# UI 蓝图

本文描述 PureLeaf 应该长成什么样，不描述具体 Qt 实现细节。

## Home

用途：工作区启动器。

- 左侧：Logo、开始写作提示、新建空白项目、打开本地文件夹。
- 右侧：最近项目列表。
- 无最近项目时：隐藏右侧区域，左侧内容居中显示。
- 最近项目行显示：项目名、根路径、字符数、Git 状态、更多菜单。

## Editor

用途：主要写作界面。

- 顶栏：返回、项目名、编译、同步、设置入口。
- 左栏：文件树、项目结构、搜索。
- 中间：LaTeX 编辑器。
- 右侧：PDF 预览。
- 底部：编译日志、错误列表、状态信息。

## Settings

用途：全局偏好。

- 编译器路径与参数。
- 主题、图标、窗口表现。
- 编辑器偏好。
- 关于与版本信息。

## 导航

- Home → Editor：新建、打开文件夹、打开最近项目。
- Editor → Home：返回启动页。
- 任意页面 → Settings：全局设置入口。
- Settings → Back：回到来源页；没有来源时回 Home。

## 规则

- Home 不承载编辑功能。
- Editor 不承载项目创建向导。
- Settings 不直接改项目内容。
- 页面之间通过 app layer / navigator 传递意图，不互相持有复杂状态。
